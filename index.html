Statistics is not magic! If the data isn’t good enough, no amount of statistical manipulation will bring out the results.

Welcome to Data Exploration page and basic statistical modeling.

Data Exploration is most important task in the data analysis; If you did analysis without it. It means that you still missing something or you have still the opportunity to improve your model.

This R function help you to generate different plot and you can save them into the pdf. Also this is test document. we draw the histrogram, boxplot. This function recolonize data type automatically and then it will plot corresponding plot.

following are arguments in R function

Data_charcteristic it is function for ploting; variable : It is variable name on which you need to plot dataIn : data frame object RespvaR : It is dependent/target variable name ; default is null so plot will be based on full data and single color. color is specifies the distribution of target variable. target variable should be binary type.
save_option : You can save the all plots either in pdf or jpeg/png formate. ploting_path : Defult path is getwd() but you can set any path where you need to store the plots. Defult_idvars: It is those variable which are key variables like id, customer id; such variables are not useful in analysis.
